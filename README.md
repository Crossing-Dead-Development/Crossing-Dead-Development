## 👋 關於我們

哈囉，這裡是 **跨越死亡（Crossing Dead Development，簡稱 CDD）**。

目前是我的個人企劃團隊，由我 **EvansGoethe** 一人經營。這個組織彙整了我正在進行的所有專案——從遊戲擴展內容、社群機器人，到未來的獨立遊戲。

## 🎮 目前主要內容

- **Minecraft 擴展內容開發**
  - 模組（Mod）
  - 插件（Plugin）
  - 資源包（Resource Pack）
- **Discord 機器人開發**
  - 社群工具與機器人

## 🚀 未來計畫

- 獨立遊戲開發
